Learn to structure code in MCV Pattern 


What is MCV pattern?
M = Model 
    DataBase --> Collection of Schema 
V = View
    User Interface --> HTML
C = Controller
    Business Logic --> Mediator Between Model and View

Benefits of MCV Pattern 
    Separation of Concerns
    Code organization
    Independent Blocks
    Reduce of Complexity of Web App
    Easy to Maintain
    Easy to Modify
    Code Resuability
    Improved Collaboration
    Platform independence
